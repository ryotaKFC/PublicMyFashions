# My Fashion

### 概要
- スマホ向けwebアプリ
- 今日のおすすめの服が知れる
- 自分のコーデをデータベースでまとめてくれる

### 機能
- **自分のコーデをまとめてくれる（データベース）**
    1. 朝　カメラで全身を撮る
    2. AIでそのコーデの属性（色、カジュアル度、季節感）などを評価
    3. それらを「マイコーデデータベース」へ登録
    4. 登録前にユーザーが細かく属性調整できるようにもする
- **今日のおすすめのコーデを表示**
    1. ユーザーが「気分、行先」を入力
    2. 天気、季節も考慮して、おすすめのコーデを表示
